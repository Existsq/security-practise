## Tech Stack
* **Language**: Java 21
* **Build**: Maven (`pom.xml`)
* **Frameworks/Libraries**  
  * Spring Boot 3.5.3  
  * Spring Security (stateless, JWT-based)  
  * Spring Data JPA + PostgreSQL driver  
  * Spring Cloud Netflix Eureka client  
  * Lombok (code generation)  
  * jjwt 0.12.6 (JWT parsing/signing)
